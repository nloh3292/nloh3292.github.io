Privacy-Preserving Fingerprint Recognition System

By Mansoor Syed, Nathan Loh, and Nikolaus Schultze

Introduction

This project aims to address the issues current passwords face, such as Inconvenience, limited Security and Accuracy, and design a system that utilizes Fingerprints as passwords or second authenticators. We want to address these issues by creating a Convolutional Neural Network for Fingerprint Recognition, allowing the model to learn and detect fingerprints and be capable of making an accurate prediction on whom the Fingerprint belongs to. In addition, we want to build a simple application that implements a Fingerprint Biometric Authentication system that lets the user "scan" to access a file.
